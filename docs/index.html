JSON validation schema for accessibility audits

Type: object

This schema is used for RGAA, RAWeb and RAAM audits, in order to be published on the Digital accessibility observatory

root auditor
Type: string

Name of the auditor. Can be left as 'Non renseigné'

root context
Type: string

Please specify here which body requested the audit, which company carried it out and whether it is an initial audit or a second control audit. Can be left as 'Non renseigné'

root audited_at
Type: stringFormat: date

The date the audit was completed. Date format: yyyy-mm-dd

root inventory
Type: object

root inventory name
Type: string

The name of the inventory item is used here as an identifier. It must therefore be copied exactly from the Excel file present on the Open data: https://data.public.lu/fr/datasets/inventaire-des-sites-publics/

root platform
Type: object

root platform name
Type: enum (of string)

One of the three keywords: Web, iOS or Android

Must be one of:

  • "Web"
  • "iOS"
  • "Android"

root audit_reference
Type: object

root audit_reference name
Type: enum (of string)

One of the three keywords: RGAA, RAWeb, RAAM

Must be one of:

  • "RGAA"
  • "RAWeb"
  • "RAAM"

root audit_reference version
Type: string

Version of the framework (1 or 1.1 for RAWeb or RAAM)

root control_type
Type: object

root control_type name
Type: enum (of string)

One of the three keywords: Complet, Mobile, Simplifié

Must be one of:

  • "Complet"
  • "Mobile"
  • "Simplifié"

root assessed_level
Type: object

root assessed_level name
Type: enum (of string)

One of the two keywords: A, AA

Must be one of:

  • "A"
  • "AA"

root pages
Type: array
No Additional Items

Each item of this array must be:

root pages pages items
Type: object

root pages pages items number
Type: integer

Page or screen number between 1 and 20

Value must be greater or equal to 1 and lesser or equal to 20

root pages pages items uri
Type: string

URI of the page. For mobile audit, title or short description of the screen

root pages pages items assessments
Type: array
No Additional Items

Each item of this array must be:

root pages pages items assessments crit
Type: object

root pages pages items assessments assessments items criterion
Type: object

root pages pages items assessments assessments items criterion number
Type: string
Must match regular expression: ^\d{1,2}\.\d{1,2}$

root pages pages items assessments assessments items status
Type: object

root pages pages items assessments assessments items status name
Type: enum (of string)

Must be one of:

  • "C"
  • "NC"
  • "NA"
  • "NT"

root pages pages items assessments assessments items exemption
Type: boolean

false by default, true if the criterion can be exempted
